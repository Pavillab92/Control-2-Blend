# Control 2 Blend

Hello, and thank you for downloading CONTROL 2 BLEND!

Control 2 Blend is a Python tool for Autodesk Maya that allows you to quickly connect
NURBS controllers to blend targets directly, using Set Driven Keys or Remap Value nodes.
It will help you reduce the time it normally requires when linking controllers to blend targets.

This tool is the result of the 10-week intensive course "Python for DCC" by Alexander Richter.
The course covers Python fundamentals, and applies those skills in a DCC
(Maya, 3ds Max, Houdini, Nuke) to build production-ready tools.

# FEATURES
* Connect NURB Controllers to Blend Targets in a few clicks.
* Supports both Set driven key and Remap value node connections.
* Built for character rigging workflows.

# INSTALLTION

1. Drag the script file into your Maya scripts folder:
maya/your version/scripts
2. Drag the image and shelf icon files into:
maya/your version/prefs/icons
3. Run the following command in Maya’s Python Script Editor:

import ctrl2blend; ctrl2blend.load_ui()

* Optional:
Middle-click drag the command to your shelf and assign the provided icon.
